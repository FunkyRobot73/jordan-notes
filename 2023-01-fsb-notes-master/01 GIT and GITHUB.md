# GIT Commands
| cmd                                       | action                                                |
| ----------------------------------------- | ----------------------------------------------------- |
| git init                                  | creates new git repo                                  |
| git status                                | show us the repo status                               |
| git add <file-to-add>                     | Adds files to staging area                            |
| git commit -m '<detailed commit message>' | Commits file to LOCAL REPOSITORY (hidden .git folder) |
